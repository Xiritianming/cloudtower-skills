# CloudTower Skills

[English README](./README.md)

推荐使用 [`vercel-labs/skills`](https://github.com/vercel-labs/skills) 安装。

查看可安装的 skill：

```bash
npx skills add smartxworks/cloudtower-skills --list
```

安装全部 skills：

```bash
npx skills add smartxworks/cloudtower-skills -a <agent>
```

安装 `metric-lookup`：

```bash
npx skills add smartxworks/cloudtower-skills --skill metric-lookup -a <agent>
```
